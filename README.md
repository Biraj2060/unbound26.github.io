# UNBOUND 26 Landing Page

A responsive React landing page for PACE × ACEM's UNBOUND 26 event.

## Run locally

```bash
npm install
npm run dev
```

## Registration links

The Ideathon and E-Sports Google Form links are configured near the top of `src/App.tsx`. Update `registrationFee` there before publishing if needed.

## Included

- Responsive futuristic event landing page
- Animated registration cards and feature cards
- FAQ accordion
- Click-to-call contact links
- Floating event-choice modal
- Social, email, SEO, and mobile navigation support
